Meet app

About:

This app is a progressive web app that uses the Google Calendar API to fetch upcoming events of Career Foundry by city.

Goals:

A serverless, progressive web application (PWA) is built with React using a test-driven development (TDD) technique. It uses the Google Calendar API to get the infromation.

Features:

  • Filter events by city.
  • Show/hide event details.
  • Specify number of events.
  • Use the app when offline.
  • Add an app shortcut to the home screen.
  • View a chart showing the number of upcoming events by city.

Technologies:

The following technologies are used:
  • HTML5
  • CSS3
  • JavaScript ES6
  • Node.js
  • React
  • TDD Jest
  • Serverless function AWS-Lambda
  • API Google Calendar
